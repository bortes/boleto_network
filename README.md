# boleto_network

exemplo de uso do _hyperledger composer_.

## objeto

validar a construção de regras, lógicas para o mesmo e permissionamento.

>
> CONSIDERAMOS APENAS REGRAS SIMPLES SEM LEVAR EM CONSIDERAÇÃO OS ESTADOS `NAO_REGISTRADO` e `PENDENTE`.
>

## geração do BNA

comando para geração do BNA.

```bash
composer archive create --sourceType dir --sourceName . -a boleto_network@0.0.1.bna
```
